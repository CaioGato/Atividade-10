  • Como funciona o post?

    • O método POST cria um par nome/valor que são passados no corpo da mensagem de pedido HTTP.

    O método HTML padrão é o GET, se você quiser usar POST você tem que especificar isso no atributo method.

    Assim a requisição GET é enviada como string anexada a URL, e a requisição POST é encapsulada junto ao corpo da requisição HTTP e não podendo ser visualizada.
